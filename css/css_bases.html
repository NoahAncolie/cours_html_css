Structure d'un élément HTML ( la div )

Les div ainsi que chaque élément html ont :

  • Une marge (margin). L'espace en dehors de la boite par rapport aux autres éléments.
  • Une bordure (border). Qui entoure la boite
  • Un padding. Un espace entre le contenu de la boite et la bordure
  • Le contenu, l'intérieur de la boite, étant d'une taille ou d'une autre.

Le Margin est l'espace sur les cotés.
La Border est en rouge.
Le Padding est l'espace entre le texte et la bordure (border).

Cibler un élément HTML

Pour pouvoir décorer un élément HTML au travers d'une feille CSS, il faut pouvoir choisir quel élément modifier. Soit le cibler.

Les façons de cibler ont donc plusieurs degrés d'importance ⚠️ :

  1. Le nom de l'élément. ⚠️
  2. La class de l'élement. ⚠️⚠️
  3. L'identifiant (id) de l'élément. ⚠️⚠️⚠️


⚠️ Le ciblage par nom est le plus général :

p {
  color: green;
}

Colorera tout le texte contenu dans des balises p en vert

⚠️⚠️ Par dessus le ciblage par nom se trouve le ciblage par class:

.mon_texte_rouge {
  color: red;
}

Colorera tout le texte contenu dans des balises avec class="mon_texte_rouge" en rouge

⚠️⚠️⚠️ Et tout en haut de la pyramide se trouve le ciblage par ID:

#mon_texte_bleu {
  color: blue;
}

Colorera tout le texte contenu dans des balises avec id="mon_texte_bleu" en bleu

Notez qu'en CSS, les classes sont identifiés par un point et les id par une dièse (. #).
De plus, une règle css peut être mise en importance en ajoutant !important derrière la régle.

.mon_texte_orange {
  color: orange !important;
}
